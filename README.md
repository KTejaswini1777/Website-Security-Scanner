# Website Security Scanner

## Description
The Website Security Scanner is a simple web-based application developed using Flask.  
It analyzes a given website URL and provides basic security insights such as HTTPS usage, response time, and HTTP headers.

---

## Objectives
- To analyze website security parameters  
- To identify whether a website uses HTTPS (secure communication)  
- To measure response time and server behavior  

---

## Features
- Input any website URL  
- Check HTTP status code  
- Detect HTTPS (secure / not secure)  
- Measure response time  
- Display server headers  

---

## Technologies Used
- Python  
- Flask  
- HTML, CSS  
- Requests Library
